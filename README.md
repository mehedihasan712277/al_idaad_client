# 🛍️ Al Idaad Client

A modern and responsive e-commerce frontend built with **Next.js**, **React**, and **Tailwind CSS**. The application provides customers with a fast, intuitive, and seamless shopping experience across desktop and mobile devices.

---

## 🚀 Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

---

## ✨ Features

- 🏠 Modern Landing Page
- 📦 Product Browsing
- 🔍 Product Details
- 🛒 Shopping Cart
- 📱 Fully Responsive Design
- ⚡ Fast Page Navigation
- 🎠 Interactive Product Sliders
- 🔔 Toast Notifications
- 📝 Secure HTML Content Rendering

---

## 📦 Dependencies

- Next.js
- React
- TypeScript
- Tailwind CSS
- Swiper
- React Hot Toast
- DOMPurify

---

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/al_idaad_client.git
```

```bash
cd al_idaad_client
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```

---

## 🛠 Available Scripts

```bash
npm run dev      # Start development server

npm run build    # Build for production

npm start        # Start production server

npm run lint     # Run ESLint
```
